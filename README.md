### Clone the repository
```bash
git clone https://zachvaughn.github.io/project-management-system-4663.git
cd project-management-system-4663
```

### Run locally
No installation or build step required. Simply open `index.html` in your browser.

> All data is stored in the browser's localStorage. No backend or database is needed.
